# 🧮 Calculadora com Interface Gráfica em Python

Aplicação desktop desenvolvida em Python utilizando a biblioteca Tkinter para criação de interface gráfica.  
O projeto simula uma calculadora funcional com operações básicas e layout personalizado.

---

## 📌 Sobre o Projeto

Esta calculadora foi desenvolvida com o objetivo de praticar:

- Manipulação de interface gráfica (GUI)
- Organização de código em funções
- Uso de variáveis globais
- Eventos e callbacks
- Estruturação visual com Frames
- Boas práticas de lógica de programação

O layout foi estilizado manualmente com paleta de cores personalizada.

---

## 🛠 Tecnologias Utilizadas

- Python 3.x
- Tkinter
- ttk (widgets avançados)
- Git
- GitHub

---

## ✅ Funcionalidades

- Operações matemáticas básicas (+, -, *, /)
- Porcentagem (%)
- Números decimais
- Botão de limpar (C)
- Atualização dinâmica do display
- Interface organizada em display e teclado

---

## 🎨 Interface

A aplicação possui:

- Display superior para exibição de valores
- Teclado organizado em grade
- Cores personalizadas
- Layout fixo e responsivo dentro da janela
